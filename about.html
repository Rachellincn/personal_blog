// 关于这个站

这是一个私人学习笔记站点。记录在学习物理课程过程中的推导、物理图像和个人理解， 不保证严格性，但尽量追求"讲清楚为什么"。

目前主要覆盖：数学物理方程（偏微分方程）、理论力学、电动力学。

// 技术栈

纯静态 HTML + CSS。公式渲染使用 KaTeX。 部署在 Vercel。

没有框架，没有 npm，没有 build step。
